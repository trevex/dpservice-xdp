# M4 — NAT Gateway Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add dpservice-style network NAT (PAT) — a guest's egress to an *external* destination is SNAT'd to a configured NAT IP + an allocated source port, pinned in conntrack so the return traffic is reverse-translated back to the guest — plus `CreateNat`/`GetNat`/`DeleteNat`/`ListLocalNats` gRPC.

**Architecture:** A guest carries a NAT config `(vni, guest_ipv4) → {nat_ipv4, port_min, port_max}`. On `guest_tx`, when the destination resolves to a route flagged **external**, the datapath allocates a source port (hash of the flow into the range + a short linear probe for a free slot — faithful to dpservice `dp_find_new_port`), rewrites src IP→nat_ip and L4 src-port→nat_port (with checksum fix-ups), and inserts a **forward** conntrack entry (so subsequent packets reuse the port) and a **reverse** conntrack entry. On `uplink_rx`, a packet to `nat_ip:nat_port` is matched in the reverse conntrack and un-NAT'd back to the guest. ICMP uses the echo **identifier** as the L4 port (rewritten like dpservice's `dp_change_icmp_identifier`).

**Tech Stack:** Rust + aya/aya-ebpf (rustup `nightly-2026-01-15`, bpf-linker), tonic gRPC, `env/netns-e2e.sh`.

**Spec:** `docs/superpowers/specs/2026-06-17-datapath-feature-parity-design.md` (milestone M4).

**Starting point (M1+M2+M3 complete):** map-driven datapath. `egress::try_guest_tx` (PORT_META → ARP → `lb::ct_reverse_snat` → `vip::snat_egress` → ROUTES → `encap`), `ingress::try_uplink_rx` (decap-prep → VIP target resolve → `lb::lb_select_dnat` → INTERFACES → write inner eth → `vip::dnat_ingress` → redirect). Maps: PORT_META/INTERFACES/ROUTES/LOCAL/VIPS/LB/MAGLEV/CONNTRACK. `csum::csum_replace4(check,&old:&[u8;4],&new:&[u8;4])->u16` (RFC-1624, host-tested) in `xdp-dp-common::csum`, re-exported by `xdp-dp-ebpf/src/csum.rs`. `parse::l4_ports(data,data_end,ip_off)->Option<(u8 proto,u16 sport,u16 dport)>` (ICMP returns `(1,id,id)`) and `parse::hash5(&src,&dst,sport,dport,proto)->u32`. gRPC `Control` owns the Ebpf + map handles + `by_id: HashMap<Vec<u8>,(u32,[u8;4])>`. Single-tenant `vni=0`.

## Design decisions locked for M4 (faithful to dpservice, see `src/dp_nat.c` / `src/nodes/snat_node.c`)

- **External trigger:** network SNAT fires when the guest has a NAT config **and** the egress destination resolves to a route flagged `is_external` — the XDP analog of dpservice's `df->flow_type == DP_FLOW_SOUTH_NORTH`. Overlay peer routes are `is_external=0`, so guest-to-guest is never NAT'd.
- **Port allocation:** `start = hash5(5-tuple) % range; nat_port = port_min + ((start + i) % range)` for the first `i` whose **reverse** conntrack key is free (bounded probe, `PROBE_LIMIT = 64`). Mirrors dpservice's hash-start + linear-probe in `dp_find_new_port`; the reverse conntrack stands in for dpservice's portoverload table. Per-flow consistency comes from the **forward** conntrack entry (reuse the same `nat_port` for subsequent packets), standing in for dpservice's `portmap` table.
- **Conntrack:** a dedicated `NAT_CT` `LRU_HASH` keyed by the 5-tuple (`CtKey`, already defined in M3), value `NatCtVal{ipv4,port}`. Two entries per new flow:
  - **reverse** `{src=ext_ip, dst=nat_ip, src_port=ext_l4, dst_port=nat_port, proto} → {guest_ip, guest_l4}` (consumed by ingress DNAT).
  - **forward** `{src=guest_ip, dst=ext_ip, src_port=guest_l4, dst_port=ext_l4, proto} → {nat_ip, nat_port}` (consumed by egress to reuse the port).
  For ICMP the reverse key's `src_port` is `nat_port` (the reply echoes our rewritten id), so the ingress lookup `(nat_port,nat_port)` matches. LRU auto-eviction suffices for the PoC; a userspace idle-timeout GC is a documented follow-on.
- **Checksums:** IP header checksum via `csum_replace4` on the address change (always). TCP/UDP L4 checksum gets **two** incremental updates — the address change (`csum_replace4`) and the port change (`csum_replace4` with the 2 high bytes zeroed in both old/new, which makes it a 16-bit replace). ICMP: the address change does **not** affect the ICMP checksum (no pseudo-header), but the **identifier** change does, so update the ICMP checksum for the id only.
- **Scope:** single-node NAT only. `CreateNeighborNat` (distributed return) stays a stub returning OK. A guest having *both* a VIP and a NAT config is out of scope (the lab never configures both on one guest).

- [ ] **Step 1: Implement `nat_snat_egress` in a new `xdp-dp-ebpf/src/nat.rs`**

```rust
use aya_ebpf::programs::XdpContext;
use xdp_dp_common::{CtKey, NatCtVal, NatKey};

use crate::csum::csum_replace4;
use crate::maps::{NAT, NAT_CT};
use crate::parse::{hash5, l4_ports};

const IPPROTO_ICMP: u8 = 1;
const IPPROTO_TCP: u8 = 6;
const IPPROTO_UDP: u8 = 17;
const PROBE_LIMIT: u16 = 64;

/// Incrementally fold a 16-bit field change (network-order `old`/`new`) into an L4/ICMP checksum
/// by reusing `csum_replace4` with the upper 2 bytes zeroed in both arguments.
#[inline(always)]
fn csum_replace2(check: u16, old: u16, new: u16) -> u16 {
    let o = old.to_be_bytes();
    let n = new.to_be_bytes();
    csum_replace4(check, &[o[0], o[1], 0, 0], &[n[0], n[1], 0, 0])
}

/// Egress network SNAT. If `is_external` and the guest (vni, src) has a NAT config, allocate a
/// source port (reusing the forward-conntrack port if the flow is already tracked), rewrite
/// src IP -> nat_ip and the L4 src port / ICMP id -> nat_port (+checksums), and pin forward +
/// reverse conntrack. Returns true if the packet was NAT'd.
#[inline(always)]
pub fn nat_snat_egress(ctx: &XdpContext, ip_off: usize, vni: u32, is_external: bool) -> bool {
    if !is_external {
        return false;
    }
    let data = ctx.data();
    let data_end = ctx.data_end();
    if data + ip_off + 20 > data_end {
        return false;
    }
    let p = data as *mut u8;
    let src = unsafe { core::ptr::read_unaligned(p.add(ip_off + 12) as *const [u8; 4]) };
    let dst = unsafe { core::ptr::read_unaligned(p.add(ip_off + 16) as *const [u8; 4]) };
    let nat = match unsafe { NAT.get(&NatKey { vni, ipv4: src }) } {
        Some(v) => *v,
        None => return false,
    };
    let range = nat.port_max.wrapping_sub(nat.port_min);
    if range == 0 {
        return false;
    }
    let (proto, sport, dport) = match l4_ports(data, data_end, ip_off) {
        Some(v) => v,
        None => return false,
    };

    // Forward conntrack: reuse the allocated port for an already-tracked flow.
    let fwd_key = CtKey {
        src_ip: src,
        dst_ip: dst,
        src_port: sport,
        dst_port: dport,
        proto,
        _pad: [0; 3],
    };
    let nat_port = match unsafe { NAT_CT.get(&fwd_key) } {
        Some(v) => v.port,
        None => {
            // Allocate: hash the flow to a start slot, linear-probe for a free reverse key.
            let start = (hash5(&src, &dst, sport, dport, proto) % range as u32) as u16;
            let mut chosen = nat.port_min.wrapping_add(start);
            let mut i: u16 = 0;
            while i < PROBE_LIMIT {
                let cand = nat.port_min.wrapping_add((start.wrapping_add(i)) % range);
                // For ICMP the reply echoes our rewritten id, so the reverse src_port is the
                // nat_port too; for TCP/UDP it is the unchanged ext (original dst) port.
                let rev_src_port = if proto == IPPROTO_ICMP { cand } else { dport };
                let rev_key = CtKey {
                    src_ip: dst,
                    dst_ip: nat.nat_ipv4,
                    src_port: rev_src_port,
                    dst_port: cand,
                    proto,
                    _pad: [0; 3],
                };
                if unsafe { NAT_CT.get(&rev_key) }.is_none() {
                    chosen = cand;
                    let _ = NAT_CT.insert(&rev_key, &NatCtVal { ipv4: src, port: sport }, 0);
                    break;
                }
                i += 1;
            }
            let _ = NAT_CT.insert(
                &fwd_key,
                &NatCtVal {
                    ipv4: nat.nat_ipv4,
                    port: chosen,
                },
                0,
            );
            chosen
        }
    };

    // Rewrite src IP guest -> nat_ip (+ IP checksum), then the L4 src port / ICMP id -> nat_port.
    let ihl = (unsafe { *p.add(ip_off) } & 0x0f) as usize * 4;
    unsafe {
        core::ptr::write_unaligned(p.add(ip_off + 12) as *mut [u8; 4], nat.nat_ipv4);
        let ipc = u16::from_be(core::ptr::read_unaligned(p.add(ip_off + 10) as *const u16));
        core::ptr::write_unaligned(
            p.add(ip_off + 10) as *mut u16,
            csum_replace4(ipc, &src, &nat.nat_ipv4).to_be(),
        );
        let l4 = ip_off + ihl;
        if proto == IPPROTO_TCP && data + l4 + 18 <= data_end {
            let c0 = u16::from_be(core::ptr::read_unaligned(p.add(l4 + 16) as *const u16));
            let c1 = csum_replace4(c0, &src, &nat.nat_ipv4);
            let c2 = csum_replace2(c1, sport, nat_port);
            core::ptr::write_unaligned(p.add(l4 + 16) as *mut u16, c2.to_be());
            core::ptr::write_unaligned(p.add(l4) as *mut u16, nat_port.to_be());
        } else if proto == IPPROTO_UDP && data + l4 + 8 <= data_end {
            let c0 = u16::from_be(core::ptr::read_unaligned(p.add(l4 + 6) as *const u16));
            if c0 != 0 {
                let c1 = csum_replace4(c0, &src, &nat.nat_ipv4);
                let c2 = csum_replace2(c1, sport, nat_port);
                core::ptr::write_unaligned(p.add(l4 + 6) as *mut u16, c2.to_be());
            }
            core::ptr::write_unaligned(p.add(l4) as *mut u16, nat_port.to_be());
        } else if proto == IPPROTO_ICMP && data + l4 + 8 <= data_end {
            // ICMP checksum at l4+2, identifier at l4+4. Address change does not affect it.
            let c0 = u16::from_be(core::ptr::read_unaligned(p.add(l4 + 2) as *const u16));
            let c1 = csum_replace2(c0, sport, nat_port);
            core::ptr::write_unaligned(p.add(l4 + 2) as *mut u16, c1.to_be());
            core::ptr::write_unaligned(p.add(l4 + 4) as *mut u16, nat_port.to_be());
        }
    }
    true
}
```

```rust
/// Ingress reverse DNAT. If the packet's (src, dst, ports) matches a reverse NAT conntrack entry
/// (i.e. it is the return of a NAT'd flow to nat_ip:nat_port), restore dst IP -> guest_ip and the
/// L4 dst port / ICMP id -> guest port (+checksums), and return Some(guest_ip) for delivery.
#[inline(always)]
pub fn nat_dnat_ingress(ctx: &XdpContext, ip_off: usize) -> Option<[u8; 4]> {
    let data = ctx.data();
    let data_end = ctx.data_end();
    if data + ip_off + 20 > data_end {
        return None;
    }
    let p = data as *mut u8;
    let src = unsafe { core::ptr::read_unaligned(p.add(ip_off + 12) as *const [u8; 4]) };
    let dst = unsafe { core::ptr::read_unaligned(p.add(ip_off + 16) as *const [u8; 4]) };
    let (proto, sport, dport) = l4_ports(data, data_end, ip_off)?;
    let key = CtKey {
        src_ip: src,
        dst_ip: dst,
        src_port: sport,
        dst_port: dport,
        proto,
        _pad: [0; 3],
    };
    let val = match unsafe { NAT_CT.get(&key) } {
        Some(v) => *v,
        None => return None,
    };
    let guest_ip = val.ipv4;
    let guest_port = val.port;
    let ihl = (unsafe { *p.add(ip_off) } & 0x0f) as usize * 4;
    unsafe {
        core::ptr::write_unaligned(p.add(ip_off + 16) as *mut [u8; 4], guest_ip);
        let ipc = u16::from_be(core::ptr::read_unaligned(p.add(ip_off + 10) as *const u16));
        core::ptr::write_unaligned(
            p.add(ip_off + 10) as *mut u16,
            csum_replace4(ipc, &dst, &guest_ip).to_be(),
        );
        let l4 = ip_off + ihl;
        if proto == IPPROTO_TCP && data + l4 + 18 <= data_end {
            let c0 = u16::from_be(core::ptr::read_unaligned(p.add(l4 + 16) as *const u16));
            let c1 = csum_replace4(c0, &dst, &guest_ip);
            let c2 = csum_replace2(c1, dport, guest_port);
            core::ptr::write_unaligned(p.add(l4 + 16) as *mut u16, c2.to_be());
            core::ptr::write_unaligned(p.add(l4 + 2) as *mut u16, guest_port.to_be());
        } else if proto == IPPROTO_UDP && data + l4 + 8 <= data_end {
            let c0 = u16::from_be(core::ptr::read_unaligned(p.add(l4 + 6) as *const u16));
            if c0 != 0 {
                let c1 = csum_replace4(c0, &dst, &guest_ip);
                let c2 = csum_replace2(c1, dport, guest_port);
                core::ptr::write_unaligned(p.add(l4 + 6) as *mut u16, c2.to_be());
            }
            core::ptr::write_unaligned(p.add(l4 + 2) as *mut u16, guest_port.to_be());
        } else if proto == IPPROTO_ICMP && data + l4 + 8 <= data_end {
            let c0 = u16::from_be(core::ptr::read_unaligned(p.add(l4 + 2) as *const u16));
            let c1 = csum_replace2(c0, dport, guest_port);
            core::ptr::write_unaligned(p.add(l4 + 2) as *mut u16, c1.to_be());
            core::ptr::write_unaligned(p.add(l4 + 4) as *mut u16, guest_port.to_be());
        }
    }
    Some(guest_ip)
}
```
